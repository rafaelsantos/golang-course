### Prerequisites

To execute the project hosted in this repository, first configure all dependencies

```sh
$ cd $GOPATH/src/
$ git clone https://github.com/rafaelsantos/golang-course.git
$ cd golang-course
$ go get
```

Running project

```sh
$ go run main.go
```
